# ResearchPlot-MATLAB
This repository presents MATLAB scripts used in creating scientific plots, suitable for research and presentation purposes.

## Repository Structure

- `/src`: Contains the MATLAB source code for generating plots.
- `/figures`: Contains the SVG files generated by the MATLAB scripts.
- `/docs`: Contains documentation and additional information.

## Getting Started

To use these scripts, you'll need MATLAB installed on your system. The scripts are developed and tested with **MATLAB version R2024a**. Please ensure you have this version or a compatible one to avoid any compatibility issues.

Some scripts may require original data files to function properly. Please refer to the comments within each script for details on required data and how to obtain it.
